GENERAL
The goal of this for is to have a build tailored to be used in my projects.

CONFIGURATION
To see the configuration used when generating the solution check cmake_config_1.PNG and cmake_config_2.PNG in the root folder.
Built with visual studio 2017. 
Platform toolset v141.
Windows sdk version 10.0.16299.0.
Release and Debug x64 configs.
Static library.

INSTALL
The build ready to use is in folder install_x64. Just copy it to the third party libraries folder and rename it to assimp.
